Stephanie's Pure Data Webpage

Pure Data is a free and open source visual programming language for creating computer music.

Created by mathematician Miller Puckette in 1996, Pure Data allows you to build audio synthesis, analysis and processing applications using an interface of interconnecting objects. It is widely used by musicians, sound designers, and researchers for everything from live performance to instillation art.

Pure Data is a real-time audio processing and is completely free and open-source. Unlike other computer music software alternatives, Pure Data gives you direct access to the building blocks of digital signal processing. You can create custom synthesizers, samplers, effects processors, tuning systems, and interactive sound installations using Pure Data's visual construction kit-like set up.

The software runs on Windows, macOS, Linux, and embedded systems like Raspberry Pi, making it versatile for different projects.

With this website I wanted to create a place to showcase my own patches I've created to produce my organ music digitally, and patches I've created to utilize alternative tuning systems.
